Project Architecture & Industrial Workflow

Interactive engineering diagram illustrating real-time physical telemetry, edge AI diagnostics, and cloud synchronization pipelines.

💡 How the System Works End-to-End:

  1. Data Collection: Sensors on the pump loop (RPM, Flow FIT, Pressure PIT, Temperature, Vibration) stream telemetry to the FastAPI server.
  2. AI Diagnostics: The Autoencoder models calculate Anomaly Scores in real-time, mapping reconstruction errors to classify failures (e.g. Pump Cavitation).
  3. Edge Interlocks: If a fault triggers, the AI Safety agent generates PLC Structured Text code to execute safety overrides.
  4. Cloud Syncing: Azure DTDL models are updated dynamically via JSON patches, while DevOps REST clients submit maintenance tickets directly to remote boards.
  5. SLA Alerts: Twilio API broadcasts SMS alerts to personnel devices if the local operator fails to acknowledge alarms within 15 seconds.
